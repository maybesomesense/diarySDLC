## 2. Телефонная книга <a name="2"></a>

### 2.1 Добавить номер телефона и ФИО человека <a name="2.1"></a>
**Описание.** Добавление номера телефона и фио человека в таблицу.

**Основной поток.**
1. Переход на окно телефонной книги.
2. В данном окне имеются два поля: ФИО и номер телефона.
3. Необходимо заполнить два данных поля.
4. Следует кликнуть кнопку "Добавить".
5. Данное ФИО и номер телефона выводится в таблице.

### 2.2 Найти в таблице по ФИО <a name="2.2"></a>
**Описание.** Поиск в таблице по полю ФИО.

**Основной поток.**
1. Необходимо заполнить специальное поле для поиска ФИО.
2. Кликнуть кнопку "Найти по ФИО".
3. Увидеть в таблице поиска ФИО и номер человека.

### 2.3 Найти в таблице по номеру телефона <a name="2.3"></a>
**Описание.** Поиск в таблице по полю номера телефона.

**Основной поток.**
1. Необходимо заполнить специальное поле для поиска номера телефона.
2. Кликнуть кнопку "Найти по номеру телефона".
3. Увидеть в таблице поиска ФИО и номер человека.

### 2.4 Записать данные в базу данных <a name="2.4"></a>
**Описание.** Записать данные в базу данных.

**Основной поток.**
1. Необходимо кликнуть кнопку "Записать данные в базу данных".

### 2.5 Прочитать данные из базы данных <a name="2.5"></a>
**Описание.** Прочитать данные из базы данных.

**Основной поток.**
1. Необходимо кликнуть кнопку "Прочитать данные из базы данных".
2. Увидеть обновленную таблицу.

## 5. Задачи <a name="5"></a>

### 5.1 Добавить задачу <a name="5.1"></a>
**Описание.** Добавить задачу в таблицу задач.

**Основной поток.**
1. Выбрать поле задачи.
2. Написать строку в данное поле.
3. Кликнуть кнопку "Добавить задачу".
4. Задача отобразится в таблице задач.

### 5.2 Пометить задачу как выполненную <a name="5.2"></a>
**Описание.** Добавить выполненную задачу в таблицу выполненных задач.

**Основной поток.**
1. Выбрать поле выполненной задачи.
2. Написать строку в данное поле.
3. Кликнуть кнопку "Добавить задачу в выполненные".
4. Задача отобразится в таблице выполненных задач.

### 5.3 Записать данные в базу данных <a name="5.3"></a>
**Описание.** Записать задачи и выполенные задачи в базу данных.

**Основной поток.**
1. Кликнуть "Записать в базу данных".
2. Задачи и выполненные задачи отобразятся в базе данных.

### 5.4 Прочитать данные из базы данных <a name="5.4"></a>
**Описание.** Прочитать задачи и выполенные задачи из базы данных.

**Основной поток.**
1. Кликнуть "Чтение данных из базы данных".
2. Задачи и выполненные задачи отобразятся в таблицах.

### 5.5 Найти задачу <a name="5.5"></a>
**Описание.** Найти задачу в таблице.

**Основной поток.**
1. Ввести в поле поиска задачи.
2. Кликнуть "Найти задачу".
3. Задача отобразится в таблице найденных задач.

### 5.6 Найти выполненную задачу <a name="5.6"></a>
**Описание.** Найти выполненную задачу в таблице.

**Основной поток.**
1. Ввести в поле поиска выполенных задач.
2. Кликнуть "Найти выполенную задачу".
3. Задача отобразится в таблице найденных задач.

## 6. Желания <a name="6"></a>

### 6.1. Добавить желания: название желания и путь его достижения за определённую дату <a name="6.1"></a>
**Описание.** Добавить желание в таблицу задач.

**Основной поток.**
1. Выбрать поле "Название желания".
2. Записать название названия желания.
3. Выбрать поле "Пути достижения".
4. Записать путь достижения.
5. Кликнуть "Добавить желание".
6. Выбрать дату в календаре.
7. Информация отобразится в таблице желаний.

### 6.2. Найти желание <a name="6.2"></a>
**Описание.** Найти желание из таблицы желаний.

**Основной поток.**
1. Заполнить поле название желания.
2. Кликнуть кнопку "Найти желание"
3. Увидеть информацию в таблице найденных желаний.

### 6.3. Найти путь достижения <a name="6.3"></a>
**Описание.** Найти путь достижения из таблицы желаний.

**Основной поток.**
1. Заполнить поле путь достижения.
2. Кликнуть кнопку "Найти путь достижения"
3. Увидеть информацию в таблице найденных желаний.
